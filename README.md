# Damage Logger for SA-MP

Simple damage tracking system that logs all damage taken by players with attacker details, weapon used, hit location, and timestamp.

![Damage Logs Preview](https://raw.githubusercontent.com/MartinSAMP/Damage-Logs-SA-MP/main/sa-mp-093.png)

## Features
- Logs last 50 damage instances per player
- Shows attacker name (or "World"/"Self" for environmental/self-damage)
- Accurate body part detection including headshots
- Real-time timestamp display (HH:MM:SS)
- Automatic log rotation when limit reached
- Clean tabbed dialog interface
